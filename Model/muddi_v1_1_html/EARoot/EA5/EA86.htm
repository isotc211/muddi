<b>Surface Component Type Codelist</b><br/>Codelist of specific types of Network Surface Nodes, pertaining specifically to their aspects visible at or above grade. These may be redundant with derived network feature types and supplemented by more specific component type codelists. It allows new surface utility feature types  to be added without changing the model.<br/><br/>
